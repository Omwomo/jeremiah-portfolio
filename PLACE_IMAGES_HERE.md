# Images Folder

Place your images here with **lowercase filenames**:

- `profile.jpg`        → Profile/headshot photo
- `heat-diffusion.jpg` → Heat diffusion senior project visualization
- `thermal-sim.jpg`    → Thermal modeling simulation output
- `boeing.jpg`         → Boeing internship / aerospace photo
- `tutoring.jpg`       → Academic / tutoring photo

Then update `src/App.jsx` to enable them as described in README.md.
